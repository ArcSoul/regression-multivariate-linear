Multivariate linear regression.

## Installation

`$ npm install --save ml-regression-multivariate-linear`

## Usage

```js
import MLR from 'ml-regression-multivariate-linear';

const x = [[0, 0], [1, 1], [2, 2], [3, 3]];
const y = [[0, 0, 0], [2, 4, 6], [4, 8, 12], [6, 12, 18]];
// result is ...
```
